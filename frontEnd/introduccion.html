Introducción a TypeScript

Guía completa para comenzar con TypeScript

¿Qué es TypeScript?

TypeScript es un superset de JavaScript desarrollado por Microsoft que añade tipado estático opcional al lenguaje.

  • Compilado a JavaScript puro
  • Tipado estático opcional
  • Orientado a objetos
  • Compatible con JavaScript existente

¿Qué ventajas ofrece TypeScript frente a JavaScript puro?

  • Detección temprana de errores: Encuentra errores en tiempo de compilación
  • Mejor IntelliSense: Autocompletado y sugerencias mejoradas
  • Refactoring seguro: Cambios de código más confiables
  • Documentación automática: Los tipos sirven como documentación
  • Escalabilidad: Ideal para proyectos grandes

¿Qué extensión de archivo se utiliza para los archivos TypeScript?

.ts Archivos TypeScript estándar
.tsx Archivos TypeScript con JSX (React)
.d.ts Archivos de declaración de tipos

¿Cómo se instala TypeScript en VSCode?

Instalación Global:

npm install -g typescript

Instalación Local (proyecto):

npm install --save-dev typescript

Extensiones recomendadas para VSCode:

  • TypeScript Importer
  • Bracket Pair Colorizer
  • ESLint
  • Prettier

¿Cómo se compila un archivo TypeScript para convertirlo a JavaScript?

Comando básico:

tsc archivo.ts

Compilar y ejecutar:

tsc archivo.ts && node archivo.js

Modo watch (observar cambios):

tsc archivo.ts --watch

Compilar todo el proyecto:

tsc

Usando ts-node (ejecutar directamente):

npx ts-node archivo.ts

Ejemplo Práctico

Código TypeScript (ejemplo.ts):

function saludar(nombre: string): string {
    return `Hola, ${nombre}!`;
}

const usuario: string = "Ana";
console.log(saludar(usuario));

Después de compilar con tsc ejemplo.ts:

function saludar(nombre) {
    return "Hola, " + nombre + "!";
}

var usuario = "Ana";
console.log(saludar(usuario));
